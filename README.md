- `mobile/`: React Native based mobile application

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Mobile

```bash
cd mobile
npm install
npm start
```

## Technologies

- Backend: Node.js, Express.js
- Frontend: React Native
- Database: MongoDB
